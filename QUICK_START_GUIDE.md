## Getting Started in 3 Steps

### Step 1: Add Elements
1. Open the editor
2. Click **Elements** tab (left sidebar)
3. Click any element button:
   - **Text** - Add regular text
   - **Heading** - Add H1, H2, etc.
   - **Rectangle** - Add colored box
   - **Circle** - Add colored circle
   - **Image** - Add image from URL

### Step 2: Position Elements
1. Click and drag any element
2. Move it anywhere on the canvas
3. Release to drop

### Step 3: Style Elements
1. Click on an element to select it
2. **Style** tab opens automatically
3. Edit properties:
   - Content (for text)
   - Size, color, position
   - Delete if needed

## Need Help?

### Element Not Appearing?
- Check if it's behind another element
- Try clicking on different areas of canvas
- Check position X/Y values in Style tab

### Can't Drag Element?
- Make sure you're clicking directly on the element
- Check if drag mode is enabled
- Try clicking and holding for a moment

### Styling Not Working?
- Make sure element is selected (blue outline)
- Check Style tab is open
- Try clicking element again
